Visually Hidden

Hide elements visually while keeping them accessible to screen readers and keyboard users.

Accessibility Tip: Use these classes instead of display: none when the info is needed for Assistive Tech.

1. Screen Reader Only (.visually-hidden)

Hidden from Sight

There is a title below this text that you cannot see, but a screen reader will announce.

Hidden Section Title

<!-- Invisible to users, visible to screen readers -->
<h2 class="visually-hidden">Title for screen readers</h2>

2. Focusable Utility (.visually-hidden-focusable)

Interactive Accessibility

Click here and press Tab to reveal the hidden link.

Skip to main content
<!-- Becomes visible only when it receives keyboard focus -->
<a class="visually-hidden-focusable" href="#">Skip to content</a>

3. Container with :focus-within

Nested Activation
A container with a focusable link inside. (Tab into this area to see the whole box).

The entire container stays hidden until one of its children is focused.

<!-- The container reveals itself when any child is focused -->
<div class="visually-hidden-focusable">
  ... <a href="#">Link</a>
</div>

4. Sass Mixins

Sass Implementation

If you prefer not to use utility classes in your HTML, use the Sass mixins:

.custom-title { @include visually-hidden; }
.skip-link { @include visually-hidden-focusable; }
