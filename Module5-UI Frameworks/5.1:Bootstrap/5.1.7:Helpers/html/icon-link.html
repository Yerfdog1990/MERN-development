Icon Link

Stylized hyperlinks that automatically align icons and text with built-in hover animations.

Note: Icons are automatically sized to 1em to perfectly match your text size.

1. Basic Structure

Standard Alignment
Icon on the left Icon on the right
<!-- Use .icon-link on any <a> element -->
<a class="icon-link" href="#">
  <i class="bi bi-box"></i>
  Link Text
</a>

2. Hover Animation

Interactive States

Add .icon-link-hover to trigger a smooth transform on hover.

Hover over me
<!-- .icon-link-hover moves the icon to the right -->
<a class="icon-link icon-link-hover">...</a>

3. Styling & Custom Vars

CSS Variable Control
Green Success Link Vertical Slide
<!-- Modify behavior with inline CSS variables -->
<a style="--bs-icon-link-transform: translate3d(0, -.125rem, 0);">...</a>

4. Accessibility Guidelines

Universal Design
  • Decorative Icons: Use aria-hidden="true" so screen readers ignore them.
  • Informative Icons: Add role="img" and an aria-label to the icon itself if it adds meaning not present in the text.
