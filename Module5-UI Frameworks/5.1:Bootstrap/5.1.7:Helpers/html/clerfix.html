Clearfix

A utility to clear floated content within a container, preventing parent collapse.

1. The Problem: Floated Collapse

Without .clearfix

Notice the dashed red border (the parent) does not expand to contain the buttons.

2. The Solution: Applied Utility

With .clearfix

The blue dashed border now correctly wraps around the floated content.

<div class="clearfix">
  <button class="float-start">Left</button>
  <button class="float-end">Right</button>
</div>

3. SCSS Mixin

Developer Reference

If you are using Sass, you can include the clearfix directly into your custom selectors without adding extra classes to your HTML.

// SCSS Implementation
.custom-container {
  @include clearfix;
}
