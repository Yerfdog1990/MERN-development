Stretched Link

Turn entire components into clickable targets by expanding a nested link's click area via pseudo-elements.

Critical: The parent must have position: relative (or a transform) to contain the stretch.

3. Identifying the Containing Block

Common Issues

A stretched link stops at the first parent that has:

  • position other than static
  • A transform or perspective value
  • A filter value
