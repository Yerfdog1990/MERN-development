Vertical Rule

A vertical version of the <hr> element, designed to separate content in horizontal layouts.

How it works: VRs have a width of 1px and a min-height of 1em.

1. Scaling in Flex Containers

Adaptive Height
Left Content
Right Content

In a flex container, the .vr stretches to match the height of the sibling elements.

<!-- The .vr stretches automatically in a d-flex container -->
<div class="d-flex" style="height: 150px;">
  <div class="vr"></div>
</div>
