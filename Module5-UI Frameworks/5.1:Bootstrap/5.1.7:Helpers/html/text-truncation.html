Text Truncation

Shorten long strings of text with an ellipsis (...) to prevent layout breaking.

Requirement: Element must be display: block or display: inline-block.

1. Block-Level Truncation

Automatic Width (via Grid)
This is a very long sentence that will be cut off by the edge of its column.

The text truncates based on the col-4 width.

<!-- works on div, p, and other block elements -->
<div class="col-4 text-truncate">
  Long text content...
</div>

2. Inline-Block (Fixed Width)

Manual Max-Width
Shortened inline text that was originally much longer.
<!-- Use d-inline-block and a width to truncate spans -->
<span class="d-inline-block text-truncate" style="max-width: 150px;">
  Long text...
</span>

3. How it Works (CSS)

The Logic

The .text-truncate class applies these three critical properties:

overflow: hidden;
text-overflow: ellipsis;
white-space: nowrap;
