Ratios

Create responsive containers that maintain a consistent aspect ratio, regardless of screen size.

Pro Tip: Perfect for YouTube embeds (<iframe>) or responsive <video> elements.

1. Default Aspect Ratios

Predefined Classes
1x1
4x3
16x9
21x9
<!-- Wrap any element in a .ratio container -->
<div class="ratio ratio-16x9">
  <iframe src="..." title="Video"></iframe>
</div>

2. Custom Ratios (CSS Variables)

On-the-fly Math

Create a custom ratio (like 2x1) by setting the --bs-aspect-ratio variable.

2x1 (Custom)
<!-- Formula: (Height / Width) * 100% -->
<div class="ratio" style="--bs-aspect-ratio: 50%;">
  <div>2x1 Ratio Content</div>
</div>
