Flexbox vs CSS Grid Quick Ref
Flexbox (Default)
  • Uses .row and .col-*
  • Spacings via padding/negative margins
  • 1-Dimensional (mostly)
CSS Grid (Experimental)
  • Uses .grid and .g-col-*
  • Spacings via gap property
  • 2-Dimensional & CSS Variable driven
