Min-Width (Mobile-First) Demo

Bootstrap's default approach: Start with mobile styles and enhance for larger screens.

1. Progressive Enhancement Card

I grow and change as the screen gets wider!
/* Base styles (Default = Mobile XS) */
.min-card { background: #6366f1; padding: 1.5rem; }

/* Tablets and up (MD) */
@media (min-width: 768px) {
  .min-card { background: #8b5cf6; padding: 3rem; }
}

/* Large Desktops and up (XL) */
@media (min-width: 1200px) {
  .min-card { background: #ec4899; transform: scale(1.05); }
}

Bootstrap's Min-Width Breakpoints

  • None (XS): < 576px (Base styles)
  • sm: ≥ 576px
  • md: ≥ 768px
  • lg: ≥ 992px
  • xl: ≥ 1200px
  • xxl: ≥ 1400px
