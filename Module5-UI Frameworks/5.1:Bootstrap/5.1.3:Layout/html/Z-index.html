Bootstrap Z-index Scale

Understanding the stacking order and component layering.

1. Visual Stacking Order (3D Concept)

Dropdown (1000)
Sticky (1020)
Fixed (1030)
Modal (1055)
Tooltip (1080)
$zindex-dropdown: 1000;
$zindex-sticky: 1020;
$zindex-fixed: 1030;
$zindex-modal: 1055;
$zindex-tooltip: 1080;

2. Low-Value Z-index (Component Borders)

Focus on the input to see its border rise above the button (Z-index: 3).

// Internal Bootstrap logic
.form-control:focus {
  z-index: 3; // Renders border on top
}
