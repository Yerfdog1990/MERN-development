Bootstrap Form Controls

Detailed notes on styling <input> and <textarea> with custom styles, sizing, and states.

1. Sizing Variants

Control heights using .form-control-lg and .form-control-sm.

Control Sizes
<input class="form-control form-control-lg" type="text"> <input class="form-control form-control-sm" type="text">

2. Form Text (Helper Text)

Create block or inline hint text using .form-text.

Block-level Accessibility
Must be 8-20 characters long.
<input aria-describedby="passHelp"> <div id="passHelp" class="form-text">...</div>

3. Readonly & Plain Text

Use .form-control-plaintext to remove the border while keeping spacing.

Standard Readonly
Plaintext Readonly
<input readonly class="form-control-plaintext">

4. Specialized Inputs

Styling for type="file" and type="color".

File Input
Color Picker
<input type="file" class="form-control"> <input type="color" class="form-control form-control-color">

5. Datalists

Combining input fields with pre-defined options.

Datalist (Autocomplete)
<input list="datalistOptions"> <datalist id="datalistOptions">...</datalist>
