Floating Labels

Beautifully simple labels that animate over input fields.

1. Basic Implementation

Input + Label Pair
<div class="form-floating"> <input placeholder="Required" id="f1"> <label for="f1">Label text</label> </div>

2. Textareas (Custom Height)

<!-- Use inline style for height, not rows attribute --> <textarea style="height: 100px">...</textarea>

3. Select Menus & Disabled State

<!-- Labels for selects are always in the floated position --> <select class="form-select">...</select>

4. Nesting in Input Groups

@
<div class="input-group"> <span class="input-group-text">@</span> <div class="form-floating">...</div> </div>

5. Floating Labels in Grid

<div class="row g-2"> <div class="col-md"> <div class="form-floating">...</div> </div> </div>
