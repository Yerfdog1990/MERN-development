Bootstrap Select

Customizing native <select> elements with custom CSS and interaction states.

1. Default Custom Select

Bootstrap replaces the native arrow with a custom SVG background icon using .form-select.

Base Styling
<select class="form-select"> <option selected>Open this select menu</option> </select>

2. Sizing Variants

Use .form-select-lg or .form-select-sm to match input sizes.

Large Select
Small Select
<select class="form-select form-select-lg">...</select> <select class="form-select form-select-sm">...</select>

3. Multiple & Size Attributes

Native attributes multiple and size change the layout to a list box.

Multiple Selection
Fixed Rows (Size="3")
<select class="form-select" multiple>...</select> <select class="form-select" size="3">...</select>

4. Disabled State

Add the disabled attribute to prevent interaction and gray out the UI.

Disabled Selection
<select class="form-select" disabled>...</select>
