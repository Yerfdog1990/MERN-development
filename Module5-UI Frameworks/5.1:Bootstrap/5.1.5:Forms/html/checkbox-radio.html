Bootstrap Checks & Radios

A guide to consistent checkboxes, radios, switches, and toggle buttons.

1. Default Stacked Layout

Uses .form-check as a wrapper with .form-check-input and .form-check-label as siblings.

Checkboxes
Radios
<div class="form-check"> <input type="checkbox" class="form-check-input" id="c1"> <label class="form-check-label" for="c1">Label</label> </div>

2. Switches

Add .form-switch to the wrapper for a toggle appearance. Use role="switch" for accessibility.

Toggle Switches
<div class="form-check form-switch"> <input class="form-check-input" role="switch"> </div>

3. Inline & Reverse Layouts

Use .form-check-inline for horizontal rows or .form-check-reverse to flip alignment.

Inline Group
Reverse Alignment
<!-- Inline --> <div class="form-check form-check-inline">...</div> <!-- Reverse --> <div class="form-check form-check-reverse">...</div>

4. Toggle Buttons

Use .btn-check on the input and .btn classes on the label.

Button Style Toggles
<input type="checkbox" class="btn-check" id="b1"> <label class="btn btn-outline-primary" for="b1">Toggle</label>
