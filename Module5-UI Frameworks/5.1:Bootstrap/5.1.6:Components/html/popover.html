Popovers

Interactive overlays that house secondary information, powered by the Popper engine.

Note: Popovers are opt-in. You must initialize them via JavaScript.

1. The Basic Trigger

Attributes Required
// JS Initialization (Required)
const popoverTriggerList = document.querySelectorAll('[data-bs-toggle="popover"]');
const popoverList = [...popoverTriggerList].map(el => new bootstrap.Popover(el));

<!-- HTML Trigger -->
<button data-bs-toggle="popover" data-bs-title="Header" data-bs-content="Body">...</button>

2. Directional Placement

data-bs-placement
<button data-bs-placement="top">Top</button>
<button data-bs-placement="right">Right</button>
<button data-bs-placement="bottom">Bottom</button>
<button data-bs-placement="left">Left</button>

3. Dismiss on Next Click

Trigger: Focus Dismissible Popover

Requires an <a> tag and tabindex="0".

<!-- Must use <a> for proper cross-browser focus behavior -->
<a tabindex="0" data-bs-toggle="popover" data-bs-trigger="focus">...</a>

4. Triggering on Disabled Buttons

Wrapper Technique
<!-- Wrap disabled elements in a focusable span -->
<span class="d-inline-block" tabindex="0" data-bs-toggle="popover">
  <button class="btn" disabled>...</button>
</span>
