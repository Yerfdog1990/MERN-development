Tooltips

Small overlays of content for providing contextual information when hovering or focusing.

Essential: Tooltips are opt-in and require Popper.js to function.

1. Mandatory Initialization

JavaScript Setup

Tooltips do not work until you initialize them via JavaScript.

// Select all elements with data-bs-toggle="tooltip"
const triggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]');
const tooltipList = [...triggerList].map(el => new bootstrap.Tooltip(el));

2. Four Directions

Placement Options
<button data-bs-toggle="tooltip" data-bs-placement="top" title="...">...</button>

3. HTML & Custom Themes

Rich Content
<!-- For HTML: -->
<button data-bs-html="true">...</button>

<!-- For Custom Class (CSS): -->
.custom-tooltip { --bs-tooltip-bg: #4338ca; }

4. Disabled Elements

Accessibility Workaround

Hovering a 'disabled' attribute doesn't trigger tooltips. Wrap it in a span.

<span class="d-inline-block" tabindex="0" data-bs-toggle="tooltip">
  <button disabled>...</button>
</span>
