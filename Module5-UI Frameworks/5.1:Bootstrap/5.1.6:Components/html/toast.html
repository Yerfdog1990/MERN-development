Toasts

Push-style notifications built with flexbox. Lightweight, easily positioned, and highly customizable.

Note: Toasts are opt-in. You must initialize them with JavaScript to make them appear.

1. Basic Structure

Standard Anatomy
System Just now
Welcome back! You have 3 new messages.
<!-- Use role="alert" for assertive, role="status" for polite -->
<div class="toast" role="alert">
  <div class="toast-header">
    <strong class="me-auto">Title</strong>
    <button data-bs-dismiss="toast"></button>
  </div>
  <div class="toast-body">Message content.</div>
</div>

// Initialization Script
const btn = document.getElementById('liveToastBtn');
const element = document.getElementById('liveToast');
const toast = bootstrap.Toast.getOrCreateInstance(element);

btn.addEventListener('click', () => toast.show());
