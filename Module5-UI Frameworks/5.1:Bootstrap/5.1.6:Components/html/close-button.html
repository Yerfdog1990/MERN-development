Close Button

A generic component for dismissing content like modals and alerts.

1. Default & Disabled States

Default
Disabled

2. Dark Variant (Inverted)

Note: .btn-close-white is deprecated. Use data-bs-theme="dark" instead.

Theme Dark
<!-- Wrap in a dark theme container to invert the SVG icon --> <div data-bs-theme="dark"> <button type="button" class="btn-close"></button> </div>
