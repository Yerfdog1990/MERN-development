Accordions

Vertically collapsing lists of content with built-in transition effects.

1. Default (Single-Open)

Default Behavior

By using data-bs-parent, opening this item will automatically close any other open items in the group.

This item is hidden by default.
<!-- Use 'data-bs-parent' to ensure only one item opens at a time --> <div class="accordion-collapse collapse" data-bs-parent="#myAccordion">

2. Flush Style

No Outer Borders

The .accordion-flush class removes borders and rounded corners for a cleaner, edge-to-edge look.
<div class="accordion accordion-flush">

3. Always Open (Independent)

Independent Toggles

Omit the data-bs-parent attribute to allow multiple items to stay expanded simultaneously.

Both of these can be open at once.
<!-- Simply remove 'data-bs-parent' from the collapsible elements --> <div class="accordion-collapse collapse">
