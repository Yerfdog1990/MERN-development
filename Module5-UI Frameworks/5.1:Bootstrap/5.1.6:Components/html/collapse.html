Collapse

Toggle content visibility with smooth CSS height/width transitions.

1. Basic Vertical Collapse

Height Transition
This content is hidden by default. Collapsing an element animates the height from its current value to 0.
<button data-bs-toggle="collapse" data-bs-target="#id">Toggle</button> <div class="collapse" id="id"> <div class="card card-body">Content...</div> </div>

2. Horizontal Collapse

Width Transition
This transitions the width instead of height. Child width must be explicitly set.
<!-- Use .collapse-horizontal and set a width on the inner child --> <div class="collapse collapse-horizontal" id="id"> <div style="width: 300px;">...</div> </div>
