Scrollspy

Automatically update navigation links based on scroll position within a container.

Requirements: IDs on targets must match href on links. The container needs a height and overflow-y: scroll.

3. Vital Configuration

Options & Behavior
Attribute Effect
data-bs-root-margin Adjusts the "boundary" for activation (e.g., 0px 0px -25%).
data-bs-smooth-scroll Enables smooth jumping when links are clicked.
data-bs-threshold Controls percentage of target visible before activating.
